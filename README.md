# langton-ant
A Scala implementation of [Langton's ant](https://en.wikipedia.org/wiki/Langton%27s_ant), using scala-swing.

## Dependencies
* scala: 2.11.8
* scala-swing: 1.0.1
* sbt: 0.13.13

## How to use
In the root directory run:
```
sbt
```
Then in sbt console:
```
run
```
